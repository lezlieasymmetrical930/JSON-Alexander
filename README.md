JSON-Alexander is a browser extension that helps you read JSON in a clear way. It formats raw JSON into a tree view, so long text becomes easy to scan. It can help when you open API responses, config files, or data dumps in your browser.

Use it when you want to:

- Expand and collapse nested objects
- Search for keys and values
